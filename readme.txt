A powerful and customizable gallery block for WordPress.

== Description ==

Galleryberg Gallery Block enhances your WordPress content creation experience with a feature-rich gallery solution for the block editor. Create stunning image galleries with multiple layout options and built-in lightbox functionality.

= Key Features =

* **Multiple Gallery Layouts** - Choose from tiles, square, masonry, or justified layouts to best showcase your images
* **Responsive Design** - Adjustable column counts that adapt beautifully to any screen size
* **Built-in Lightbox** - Professional lightbox with zoom effects, keyboard navigation, and touch support
* **Advanced Image Controls** - Individual image customization including borders, captions, and aspect ratios
* **Performance Optimized** - Server-side rendering ensures fast page loads
* **Accessibility Ready** - Proper ARIA labels and keyboard navigation support

= Layout Options =

* **Tiles Layout** - Classic grid layout with consistent spacing
* **Square Layout** - Perfect square image containers
* **Masonry Layout** - Pinterest-style layout that maximizes space usage
* **Justified Layout** - Rows of images with equal heights

= Customization Options =

* Adjustable number of columns (1-8)
* Custom spacing between images
* Image cropping toggle
* Caption display options
* Border and border-radius controls
* Multiple image size options
* Duotone filter support
* Custom link destinations

= Lightbox Features =

* Smooth zoom open/close effects
* Slide navigation between images
* Keyboard navigation (arrow keys, ESC)
* Touch-friendly on mobile devices
* Loop functionality
* Image zoom capability
* Draggable images

This plugin uses [GLightbox](https://github.com/biati-digital/glightbox) for its lightbox functionality.

== Frequently Asked Questions ==

= How do I create a gallery? =

After installing the plugin, simply add the Galleryberg Gallery block to any post or page, then click "Upload" or "Media Library" to add your images.

= Can I change the number of columns? =

Yes! You can adjust the number of columns from 1 to 8 in the block settings sidebar. The gallery will automatically adapt for smaller screens.

= Does the lightbox work on mobile devices? =

Absolutely! The lightbox is fully responsive and includes touch support for swiping between images on mobile devices.

= Can I disable the lightbox? =

Yes, you can disable the lightbox feature in the block settings if you prefer images to open in a new tab or not be clickable.

= What image sizes are supported? =

The plugin supports all WordPress registered image sizes including thumbnail, medium, large, and full size. You can select your preferred size in the block settings.

= Can I add captions to images? =

Yes, each image can have its own caption. You can toggle caption display on or off for the entire gallery.

= Is the plugin compatible with page builders? =

The plugin is designed specifically for the WordPress block editor (Gutenberg) and works seamlessly with it. It may not be compatible with other page builders.

= Can I customize the gallery appearance with CSS? =

Yes, the plugin generates semantic HTML with appropriate classes that you can target with custom CSS for further customization.
